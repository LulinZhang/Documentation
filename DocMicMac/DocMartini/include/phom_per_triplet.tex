\section{Generate point hom per triplet \er{NO\_AllImTriplet}}
%
The goal of this program is to gather information about tie-points of multiplicity 3, i.e. tie-points with track length 3 and thus visible in all images of a triplet. The program \\$\er{TestLib \quad NO\_AllImTriplet}$ launches the $CPP\_GenAllImP3()$ function the defined in $cNewO\_\-PointsTriples.cpp$. Inside the $CPP\_GenAllImP3()$, for each image within the called pattern the following programme is executed in parallel \er{$TestLib \quad NO\_OneImTriplet$}.\par 
%
Then, the $TestLib \quad NO\_OneImTriplet$, calls the $CPP\_GenOneImP3()$ function (see Alg.~\ref{alg:GenTripAll}) defined in $cNewO\_PointsTriples.cpp$. Inside this function, it creates an object of class $cAppli\_Gen\-PTripleOneImage$, and launches this class' member function -- $GenerateTriplets()$ (cf. Fig.~\ref{alg:GenTripOne}).
